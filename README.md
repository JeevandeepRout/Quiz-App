# 🧠 Quiz App

A simple and interactive Quiz Application built with vanilla JavaScript. Users can answer multiple-choice questions, receive instant feedback, and view their final score at the end of the quiz. The application features a clean dark-themed interface and responsive design for a smooth user experience across devices.

---

## ✨ Features

* Multiple-choice questions with instant feedback
* Real-time score tracking
* Next question navigation
* Restart quiz functionality
* Responsive dark-themed user interface
* Easy to customize and add new questions

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)

## 🚀 Getting Started

1. Clone the repository:

```bash

```

2. Open `index.html` in any modern web browser.

No additional installation, packages, or build tools are required.
